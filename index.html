Plain Market

Cautious prediction-market opportunities for small capital

This repository contains a beginner-first MVP that scans live Polymarket books for simple, explainable market inconsistencies, defaults to paper trading, and keeps live mode heavily constrained.

Read the build notes Browse the backend code

What this product does

Finds simple setups Starts with live neg-risk baskets and clear date-ladder logic gaps on Polymarket.
Explains every idea Each recommendation says why it may work, what could go wrong, and why the app is not certain.
Prefers safety Paper trading is the default. Weak edges, stale data, and thin books are rejected.

GitHub Pages note

GitHub Pages can only host static files. The working MVP itself is a FastAPI app in backend/, so Pages can host this overview page, but not the live Polymarket backend.

To run the app, deploy the FastAPI service somewhere that supports Python, or run it locally with uvicorn app.main:app --reload from the backend folder.
